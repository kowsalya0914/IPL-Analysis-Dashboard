# IPL Analysis Dashboard (2008–2025)

## Project Overview
This project is an interactive Power BI dashboard that analyzes IPL data from 2008 to 2025.

The dashboard provides insights into:
- Team performance
- Orange Cap winners
- Purple Cap winners
- Match statistics
- Venue analysis
- Player performance
- Boundaries and wicket analysis

---

## Tools Used
- Power BI
- DAX
- Power Query
- Excel / CSV Dataset

---

## Features
- Dynamic season filter
- Interactive KPI cards
- Team-wise analysis
- Player image integration
- Orange Cap & Purple Cap tracking
- Match result insights
